Webmention endpoint

Externe platformen kunnen een Webmention sturen naar dit platform via onderstaand endpoint.

Endpoint URL

https://snowcoach-production.up.railway.app/webmention

Methode

POST met source en target als form parameters

Link tag

Voeg dit toe aan je <head> zodat clients het endpoint automatisch detecteren:

<link rel="webmention" href="https://snowcoach-production.up.railway.app/webmention" />

Voorbeeld request

POST /webmention
Content-Type: application/x-www-form-urlencoded

source=https://jouwsite.com/pagina
target=https://snowcoach-production.up.railway.app/
